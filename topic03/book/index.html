ER Diagram

This simple ER diagram for the following description: An airline ticket booking system stores data on the flights operating for each airline, each flight occurs on a particular day and has a price per seat. Each psasenger is allocated a ticket for a particular flight and their credit card details are also stored.

Open Eclipse and ensure your workspace is pointing to the correct directory where you have your existing projects stored. Create a new static web project called lab04. Create a new XML File called airline-system.xml

Create elements that will store the data shown in the flight and flight-details entities in the ER diagram. Once you create those elements make sure you have 5 flights stored in your xml file.
